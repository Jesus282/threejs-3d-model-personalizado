# FBX Viewer con Three.js

## Descripción

Este proyecto consiste en un visor de modelos 3D en formato FBX desarrollado con Three.js. Permite cargar distintos modelos con animaciones, visualizarlos en tiempo real y manipular la cámara para inspeccionar la escena desde diferentes ángulos.

Se integró una interfaz web básica con barra de navegación (navbar) y pie de página (footer), manteniendo el renderizado del canvas dentro de un contenedor estructurado.

---

## Objetivo

Desarrollar una aplicación web capaz de cargar, renderizar y animar modelos FBX, integrando controles de cámara e interfaz gráfica para interacción del usuario.

---

## Características

- Carga de modelos FBX dinámicamente
- Reproducción de animaciones incluidas en el modelo
- Control de cámara con OrbitControls
- Iluminación básica (hemisférica y direccional)
- Visualización de métricas de rendimiento (FPS)
- Interfaz gráfica para cambiar modelos (GUI)
- Layout estructurado con navbar, canvas y footer

---

## Tecnologías utilizadas

- JavaScript (ES Modules)
- Three.js
- WebGL
- HTML5
- CSS3
